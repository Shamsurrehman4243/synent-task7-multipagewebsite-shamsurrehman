# Multi-Page Website - Synent Technologies

## Project Overview
This project is a responsive multi-page website built using HTML and CSS.  
It represents a professional company website for Synent Technologies.

---

## Objective
Build a complete multi-page website with:
- Clean user interface design
- Responsive layout (mobile-friendly)
- Navigation between pages
- Consistent design across all pages


## Pages Included
- Home (index.html)
- About (about.html)
- Services (services.html)
- Contact (contact.html)


## Features
- Fully responsive design (mobile, tablet, desktop)
- Modern and clean UI layout
- Navigation bar with active page highlight
- Hero section with background image
- Services section with cards and hover effects
- Contact form with input fields
- Contact information section (email, phone, address)
- Image integration on all pages
- Reusable CSS (single style.css file)


## Technologies Used
- HTML5
- CSS3 (Flexbox and Responsive Design)

## Responsive Design
The website is optimized for:
- Mobile devices
- Tablets
- Desktop screens

Key improvements:
- Mobile-friendly navigation bar
- Flexible layouts using Flexbox
- Optimized spacing and font sizes

## How to Run
1. Download or clone the project
2. Open the folder in VS Code
3. Open index.html
4. Run using Live Server
